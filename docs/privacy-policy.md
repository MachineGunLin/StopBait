# Privacy Policy for StopBait (别想骗我点击)

Last updated: 2026-03-26

StopBait does not collect, transmit, or sell personal data.

## What data the extension uses

StopBait stores the following data locally in the user's browser via `chrome.storage.local`:

- User-defined filtering rules
- Starter-pack toggle states
- Filter counter values
- Trusted-author whitelist entries

## What StopBait does not do

- No account login
- No remote database
- No analytics or tracking SDKs
- No third-party data sharing
- No sale or transfer of user data

## Host access scope

StopBait runs only on Xiaohongshu pages (`xiaohongshu.com`) to apply on-page visual filtering according to local user settings.

## Remote code

StopBait does not use remote code execution for extension logic.
